SwapKart

Swapkart is an innovative, web-based platform built using Java Spring Boot and PostgreSQL. Through features such as JWT-based authentication, role-based access control, location-based search, user ratings and reviews, and admin moderation, the platform ensures a safe, transparent, and trustworthy environment for all users. SwapKart’s goal is to encourage eco-conscious behavior and foster a community-driven culture of responsible consumption by integrating multiple transaction types - sell, swap, and donate - within a single, efficient system.

LockChat: Secure Conversation with Triple Authentication

LockChat is an advanced chatting application designed to ensure maximum privacy and data protection through end-to-end encryption and triple-layer authentication. It uses AES-based hybrid cryptography to secure all communications, making unauthorized access nearly impossible. Users can authenticate via face recognition, One-Time Password (OTP), or password login, offering both flexibility and robust security. With its intuitive and user-friendly interface, LockChat seamlessly combines security, simplicity, and reliability, redefining safe digital communication in the modern age.

Electricity Bill Management System

The Electricity Bill Management System is a PHP-based web application designed to simplify and modernize electricity bill management for users and administrators. It provides a centralized platform where users can securely log in to view and pay their electricity bills online, track billing history, and file complaints with ease. The system also enables administrators to efficiently generate bills, manage user accounts, and monitor payment records through an intuitive dashboard. This project aims to bring convenience, accuracy, and efficiency to the electricity billing process, making it a complete digital solution for both consumers and service providers.
